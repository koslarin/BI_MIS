# BI_MIS

This work implements methods for efficient Bayesian inference for two-stage models in environmental epidemiology using two independent sampling based algorithms, IIS and AIS.

The code for our functions, AIS and IIS, in the linear model case is provided in the main folder.

The Example1_Sim and Example2_Sim files in the Simulation_Code folder contain the code that can be used to reproduce simulations from Section 4, specifically Examples 1 and 2.

Unfortunately, stage-one output data from Section 5.1 are not publicly available.
